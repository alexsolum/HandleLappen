# Plan 20-04: Admin Varekatalog Enhancement

Enhance the Admin 'Varekatalog' view to show product images and brands, and allow manual overrides.

## 🛠️ Proposed Changes

### 🛒 Admin View UI (`src/routes/(protected)/admin/items/+page.svelte`)
- Update `itemsQuery` and the list display to include `brand` and `product_image_url`.
- Add a 40x40 circular thumbnail to each item row.
- Add a brand subtitle below the item name.
- Implement "Smart Dedup" logic (hide brand if it's a substring of the product name).

### ✏️ Admin Editing
- Add input fields for `brand` and `product_image_url` to the edit form.
- Add "Tøm bilde" (Clear image) button that sets the URL to `null`.
- Show a live image preview as the user types/pastes a new URL.

## 🧪 Verification Plan

### ⚙️ Automated Tests
- Run `tests/admin.spec.ts` to ensure Varekatalog management is still functional.
- Add a test case to verify brand/image URL updates in the Varekatalog.

### 🧤 Manual Verification
- Open the Varekatalog and verify:
    - Circular thumbnails and brand subtitles are visible.
    - "Smart Dedup" works correctly.
    - Clearing an image and saving reverts it to the placeholder.
    - Updating a brand/image URL persists correctly to the database.
