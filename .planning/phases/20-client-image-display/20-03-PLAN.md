# Plan 20-03: Shopping List Item Row Enhancement

Add circular thumbnails to shopping list item rows and ensure enriched data is correctly persisted in item details.

## 🛠️ Proposed Changes

### 🖼️ Row UI (`ItemRow.svelte`)
- Update `item` prop to include `product_image_url`.
- Add a 40x40 circular thumbnail after the checkbox and before the item name.
- Use `object-cover` for the image.
- Implement shimmer skeleton while image is loading.
- Implement package icon fallback on image error (`onerror`).
- Hide the brand in the main list to save space.

### 📄 Item Details (`ItemDetailSheet.svelte`)
- Update `ItemDetailSheet` to display the product image and brand.
- Show brand as a gray subtitle below the product name.
- Hide brand if it is a substring of the product name (Smart Dedup).
- Allow editing of the brand name in the detail view.

## 🧪 Verification Plan

### ⚙️ Automated Tests
- Run `tests/lists.spec.ts` to verify basic list interactions.
- Add a new test case to check for thumbnail visibility in the list row.

### 🧤 Manual Verification
- Add an item via barcode scan (which includes an image).
- Verify the circular thumbnail appears in the shopping list.
- Open the item detail view and verify the large image and brand subtitle are present.
- Verify "Smart Dedup" logic in the detail view.
