# Plan 20-02: Barcode Scan Confirmation Sheet UI

Enhance `BarcodeLookupSheet.svelte` to show the large circular product image and brand subtitle.

## 🛠️ Proposed Changes

### 🖼️ Component UI (`BarcodeLookupSheet.svelte`)
- Update `onConfirm` callback to include `brand` and `imageUrl`.
- Add a large circular header image (48x48 px or larger) at the top of the sheet.
- Implement shimmer skeleton while image is loading.
- Implement package icon fallback on image error (`onerror`).
- Display brand as a gray subtitle below the product name.
- Apply "Smart Dedup" logic: hide brand if it's already a substring of the product name.
- Add optional brand editing field in the confirmation section.

### 📍 Route Integration (`src/routes/(protected)/lister/[id]/+page.svelte`)
- Update `handleBarcodeConfirm` to pass `brand` and `imageUrl` from the sheet to `addItemMutation`.
- Ensure `BarcodeSheetModel` data flows correctly from the mutation to the sheet.
